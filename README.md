# Sunwah Fintech iOS Prototype

这是 **Sunwah Fintech** 面向机构投资者的 iOS 投资智能助手的产品 **原型设计**（prototype design），不是正式 App 代码。

原型覆盖登录、市场情报（ETF）、组合诊断、AI 投顾、结构化产品浏览与 Admin 维护等核心流程，用于对齐产品范围、页面结构与交互，供评审与需求讨论使用。

**在线原型：** [https://sunwah-ios-prototype.vercel.app/](https://sunwah-ios-prototype.vercel.app/)

产品需求详见 [`Sunwah-Fintech-PRD-中文版.md`](Sunwah-Fintech-PRD-中文版.md)。
